## Quick Start

### Clone the repository

```bash
git clone https://github.com/brookesb91/pixel8d.git
```

### Install dependencies

```bash
npm install
```

### Launch API

```bash
nx serve api
```

### Launch Frontend

```bash
nx serve pixel8d
```
